# 球冠体育——棒球约战好助手

  "球冠体育——棒球约战好助手"是一款线上预约棒球比赛约战和了解棒球运动资讯的应用。

  主要功能:   
  
    1.首页：今日优选、比赛约战、明星场馆、最新比赛、场馆优选等;
      今日优选:优选今日数家棒球场馆，配备配备数名对战;
      比赛约战:精选各个场馆下的比赛约战列表，可直接加入约战;
      最新比赛:比赛过程中的赛事人员和赛事总比分;
    
    2.场馆：精选上海市数家棒球馆，以及棒球馆下的约战;
    
    3.资讯：关于棒球的相关资讯，国内外的棒球相关资讯;
    
    4.我的对战：全部对战、等待对战、对战结束;
